vildskud festival

Vildskud Festival is an annual independent theater festival in Copenhagen. As a two week group project we were tasked to design and develop a user-friendly, mobile-first website.
Vildskud requested for the site to be sustainable, with each page being under 0.125 mb, as well as for us to make it easier for users to get an overview of the ticket information and where to find the different venues.
Working in a small team we collaborated on designing the UI as well as the web development. Furthermore, I was responsible for creating event posters, illustrations and a new logo.
